# Profile repository publishing instructions

Create a **public** repository named exactly:

```text
Tmgilliam
```

Copy `README.md` to the repository root. GitHub displays that file on the account profile when the public repository name matches the username.

Before publishing:

- Confirm every linked repository is public and ready.
- Remove any project that is not yet available.
- Pin three to five strongest public repositories.
- Enable private-contribution visibility if desired.
- Keep profile badges restrained and evidence-backed.
